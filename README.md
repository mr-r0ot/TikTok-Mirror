# TikTok-Mirror

# ساخت نسخه آفلاین
می توانید نسخه .html آفلاین(با اینترانت ایرن کار می کند) و بدون نیاز به vpn یا پایتون ایجاد کنید و روی هر دستگاهی اجرا کنید
```
python offline.py
# chose your dir and .html file name
```
